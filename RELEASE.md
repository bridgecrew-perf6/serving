# Release 0.1.0-alpha

## Main Features

### Ascend 310 & Ascend 910 Serving Framework
* Recommended OS: Ubuntu 16.04 (or later) or EulerOS 2.5 or EulerOS 2.8
* Python version: 3.7.5
* Provide gRPC and RESTful API
* Support preprocessing & postprocessing
* Support batching
* Multiple instances
* Version Control
